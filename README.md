 🛞 Gomería  - Sistema de Gestión de Stock

Este proyecto es una aplicación web funcional desarrollada en **Python** para la gestión integral de inventario y ventas de una gomería. Permite llevar un control preciso de neumáticos por rodado, marca y modelo, automatizando el registro de ingresos y salidas de mercadería.

🚀 Funcionalidades Clave

* **Gestión de Inventario (CRUD):** Alta, baja, modificación y consulta de stock en tiempo real.
* **Registro de Ventas:** Interfaz dinámica para descontar stock automáticamente y calcular el total recaudado.
* **Persistencia de Datos:** Almacenamiento local mediante archivos CSV para garantizar que la información no se pierda al cerrar la app.
* **Buscador Inteligente:** Filtro rápido por marca, modelo o rodado para agilizar la atención al cliente.
* **Panel de Estadísticas:** Visualización del historial de ventas y métricas de recaudación total.

🛠️ Tecnologías Utilizadas

* **Python 3**
* **Streamlit:** Para la creación de la interfaz de usuario interactiva.
* **Pandas:** Para la manipulación, limpieza y procesamiento de las bases de datos.
* **OS & Datetime:** Para la gestión de archivos del sistema y registro de fechas de operaciones.
